## 🛠️ TECHNICAL SETUP

### 1. Clone & Setup

```bash
git clone https://github.com/your-username/decypher.git
cd decypher
```

### 2. Backend Setup

```bash
cd Decypher.Web
dotnet restore
dotnet run
# Backend runs on http://localhost:5000
```

### 3. Frontend Setup

```bash
cd angular-frontend
npm install
npm start
# Frontend runs on http://localhost:4200
```

### 4. Add OpenAI Key

**Create `.env` file:**
```
OPENAI_API_KEY=sk-your-key-here
```

**Or add to `appsettings.json`:**
```json
{
  "OpenAI": {
    "ApiKey": "sk-your-key-here",
    "Model": "gpt-4"
  }
}
```

### 5. Test AI Features

```bash
# Test auto-match
curl -X POST http://localhost:5000/api/agenticai/auto-match \
  -H "Content-Type: application/json" \
  -d '{"jobDescription": "Looking for .NET developer", "topN": 5}'

# You should see:
{
  "success": true,
  "matches": [
    {
      "candidateName": "Rahul Verma",
      "matchScore": 92.5,
      "reasoning": "Strong .NET background..."
    }
  ]
}
```

## 🚀 DEPLOYMENT STEPS

### Deploy Backend (Render)

```bash
1. Push to GitHub
2. Go to render.com
3. New Web Service → GitHub
4. Select repository
5. Build: dotnet publish -c Release -o out
6. Start: dotnet out/Decypher.Web.dll
7. Add environment variables:
   - ASPNETCORE_ENVIRONMENT=Production
   - ASPNETCORE_URLS=http://+:10000
   - ConnectionStrings__DefaultConnection=<SUPABASE_URL>
   - Jwt__Key=<RANDOM_48_CHARS>
   - OpenAI__ApiKey=<OPENAI_KEY>
8. Deploy!
```

### Deploy Frontend (Netlify)

```bash
1. Go to netlify.com
2. New site from Git
3. Build: cd angular-frontend && npm run build
4. Publish: angular-frontend/dist/decypher-frontend
5. Add environment variable:
   - API_URL=https://your-backend.onrender.com
6. Deploy!
```
