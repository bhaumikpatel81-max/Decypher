🚀 DEPLOYMENT IN 3 STEPS

STEP 1: Setup Database (2 minutes)
───────────────────────────────────
1. Go to https://supabase.com
2. Create new project
3. Copy connection string
   Format: postgresql://postgres:[password]@[host]:5432/postgres

STEP 2: Deploy Backend (3 minutes)
───────────────────────────────────
1. Push this code to GitHub
2. Go to https://render.com
3. New Web Service → Connect GitHub
4. Select this repository
5. Build Command: dotnet publish Decypher.Web/Decypher.Web.csproj -c Release -o out
6. Start Command: dotnet out/Decypher.Web.dll
7. Add Environment Variables:
   
   ASPNETCORE_ENVIRONMENT=Production
   ASPNETCORE_URLS=http://+:10000
   ConnectionStrings__DefaultConnection=<PASTE_SUPABASE_STRING>
   Jwt__Key=<GENERATE_RANDOM_48_CHARS>
   Jwt__Issuer=Decypher
   Jwt__Audience=DecypherUsers
   Jwt__ExpiryMinutes=1440
   
8. Click "Create Web Service"
9. Wait 5 minutes for deployment
10. Copy your backend URL (e.g., https://decypher-api.onrender.com)

STEP 3: Deploy Frontend (2 minutes)
────────────────────────────────────
1. Go to https://netlify.com
2. New site from Git → Connect GitHub
3. Build Command: cd angular-frontend && npm run build
4. Publish Directory: angular-frontend/dist/decypher-frontend
5. Add Environment Variable:
   API_URL=<YOUR_BACKEND_URL_FROM_STEP_2>
6. Deploy
7. Copy your frontend URL (e.g., https://decypher.netlify.app)

STEP 4: Final Configuration (1 minute)
───────────────────────────────────────
1. Go back to Render (backend)
2. Update AllowedOrigins environment variable:
   AllowedOrigins=<YOUR_FRONTEND_URL_FROM_STEP_3>
3. Click "Manual Deploy" → "Deploy latest commit"

❓ TROUBLESHOOTING

Backend won't start:
→ Check Render logs
→ Verify Supabase connection string
→ Ensure all environment variables are set

Frontend shows blank page:
→ Check browser console (F12)
→ Verify API_URL environment variable
→ Check backend is running (visit /health endpoint)

CORS errors:
→ Update AllowedOrigins in backend
→ Redeploy backend
→ Clear browser cache
